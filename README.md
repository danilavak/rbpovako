# rbpovako

Сервис для рекрутинга.

## Тема

Рекрутинг: вакансии, кандидаты, отклики, интервью и офферы.

## Основные сущности

- `Vacancy` - вакансия.
- `Candidate` - кандидат.
- `Application` - отклик кандидата на вакансию.
- `Interview` - интервью по отклику.
- `Offer` - предложение кандидату.

## Основные операции

- CRUD для вакансий, кандидатов, откликов, интервью и офферов.
- Отклик кандидата на вакансию.
- Планирование интервью без пересечения слотов одного интервьюера.
- Завершение интервью с обновлением статуса отклика.
- Создание оффера после завершенного интервью.
- Принятие оффера с проверкой, что кандидат не принял другой оффер.

## База данных

По умолчанию используется H2:

```text
jdbc:h2:file:./data/rbpovako
```

Параметры подключения можно переопределить через переменные окружения:

```text
DB_URL
DB_USERNAME
DB_PASSWORD
DB_DRIVER
```

## Запуск

```bash
./mvnw spring-boot:run
```

На Windows:

```powershell
.\mvnw.cmd spring-boot:run
```

## Проверка

На Windows:
